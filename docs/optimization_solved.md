## 四号谷地（Valley IV）

### バッテリー戦略比較

電力バッファ（2,000 unit/sec）を確保し、**全鉱石を100%消費**する前提での3つのバッテリー戦略を比較。

#### 戦略概要

| 戦略 | 主力バッテリー | 紫晶鉱消費方法 | 取引券レート | 評価 |
|---|---|---|--:|---|
| **A: 大容量依存** | HC Battery | 紫晶部品 | 2,170 券/min | ✓ 成立 |
| **B: 中容量依存** | SC Battery | LC併用 | **成立せず** | ✗ 電力不足 |
| **C: 小容量依存** | LC Battery | LC自体 | **2,306 券/min** | ✓ 最適 |

#### 戦略A: 大容量バッテリー依存

HCバッテリーは紫晶鉱を消費しないため、紫晶部品（源石不使用）で紫晶鉱を消費。

| 製品 | 生産レート | 源石鉱 | 紫晶鉱 | 青鉄鉱 | 電力 | 単価 |
|---|--:|--:|--:|--:|--:|--:|
| 大容量バッテリー | 18.67/min | 560 | 0 | 373 | 1,239 | 70 |
| 紫晶部品 | 240/min | 0 | 240 | 0 | 200 | 1 |
| 鋼製部品 | 353.3/min | 0 | 0 | 707 | 1,139 | 3 |
| **計** | | **560**(余剰0) | **240**(余剰0) | **1,080**(余剰0) | **2,578** | |

- 電力供給: 18.67 × 733 = 13,693 unit/sec
- 電力消費+バッファ: 2,578 + 2,000 = 4,578 unit/sec
- 電力用バッテリー: 4,578 / 733 = 6.24/min → 出荷: 12.43/min
- **取引券: 12.43×70 + 240×1 + 353.3×3 = 870 + 240 + 1,060 = 2,170 券/min**
- **全鉱石100%消費**

#### 戦略B: 中容量バッテリー依存（非推奨）

SCバッテリーは紫晶鉱を消費しないため、LCバッテリーを併用して紫晶鉱を消費する必要がある。

| 製品 | 生産レート | 源石鉱 | 紫晶鉱 | 青鉄鉱 | 電力 | 単価 |
|---|--:|--:|--:|--:|--:|--:|
| 小容量バッテリー | 48/min | 480 | 240 | 0 | 440 | 16 |
| 中容量バッテリー | 5.33/min | 80 | 0 | 53 | 76 | 30 |
| カプセルⅡ/缶詰Ⅱ | 12/min | 0 | 0 | 240 | 130 | 27 |
| **計** | | **560** | **240** | **293** | **646** | |
| **余剰 青鉄鉱** | | | | **787** | | |

**問題点:**
- 紫晶鉱消費のためにLCバッテリー48/minが必要 → 源石鉱480/min消費
- 残り源石鉱80/minではSCバッテリー5.33/minしか作れない
- 合計電力供給: 48×167 + 5.33×83 = 8,000 + 443 = 8,443 unit/sec（全量電力消費時）
- 必要電力（採掘+生産+バッファ）: 約4,000+ unit/sec
- **青鉄鉱787/minが余る → 全鉱石消費の制約を満たせない**
- **この戦略は成立しない**

#### 戦略C: 小容量バッテリー依存（推奨）

LCバッテリーは紫晶鉱を消費するため、追加製品なしで全鉱石を消費可能。

| 製品 | 生産レート | 源石鉱 | 紫晶鉱 | 青鉄鉱 | 電力 | 単価 |
|---|--:|--:|--:|--:|--:|--:|
| 小容量バッテリー | 48.0/min | 480 | 240 | 0 | 620 | 16 |
| 蕎花カプセルⅢ | 13.5/min | 0 | 0 | 540 | 927 | 70 |
| シトローム缶詰Ⅲ | 13.5/min | 0 | 0 | 540 | 927 | 70 |
| 結晶外殻 | 80/min | 80 | 0 | 0 | 33 | 1 |
| **計** | | **560**(余剰0) | **240**(余剰0) | **1,080**(余剰0) | **2,507** | |

- 電力供給: 48 × 166.67 = 8,000 unit/sec
- 電力消費+バッファ: 2,507 + 2,000 = 4,507 unit/sec
- 電力用バッテリー: 4,507 / 166.67 = 27.0/min → 出荷: 21.0/min
- **取引券: 21×16 + 13.5×70×2 + 80×1 = 336 + 1,890 + 80 = 2,306 券/min**
- **全鉱石100%消費、最高取引券レート**

#### 戦略D: HC+LC混合（最適）

HCバッテリーは電力効率が高い（733 unit/sec/個 vs LC 167 unit/sec/個）ため、電力専用に使用し、LCを全て出荷に回す戦略。さらに、発電機はバッテリーを0.25個刻みで消費できるため、必要最低限の電力消費に最適化。

| 製品 | 生産レート | 源石鉱 | 紫晶鉱 | 青鉄鉱 | 電力 | 単価 |
|---|--:|--:|--:|--:|--:|--:|
| 大容量バッテリー | 6/min | 180 | 0 | 120 | 398 | 70 |
| 小容量バッテリー | 38/min | 380 | 190 | 0 | 349 | 16 |
| 紫晶部品 | 50/min | 0 | 50 | 0 | 42 | 1 |
| 蕎花カプセルⅢ | 12/min | 0 | 0 | 480 | 184 | 70 |
| シトローム缶詰Ⅲ | 12/min | 0 | 0 | 480 | 184 | 70 |
| **計** | | **560**(余剰0) | **240**(余剰0) | **1,080**(余剰0) | **1,157** | |

- 電力消費+バッファ: 1,157 + 2,000 = 3,157 unit/sec
- 必要HC電力用: 3,157 / 733 = 4.31/min → 0.25刻みで切り上げ → **4.5/min**
- 電力供給: HC 4.5 × 733 = 3,300 unit/sec
- **電力収支: +143 unit/sec**（安全余裕）
- HC電力用: 4.5/min、HC出荷用: **1.5/min**
- LCは全量出荷（38/min）
- **取引券: 1.5×70 + 38×16 + 50×1 + 12×70×2 = 105 + 608 + 50 + 1,680 = 2,443 券/min**

#### 結論

**戦略D（HC+LC混合）が最適**。理由:

1. **電力効率の最適化** - HCは電力効率が高いため電力専用に使用
2. **LC全量出荷** - 電力をHCで賄うことでLCを全て出荷に回せる
3. **0.25刻み最適化** - 発電機の0.25個刻み消費を活用し、HC1.5/minを出荷に回す
4. **戦略Cより+137券/min** - 電力の効率的な運用により出荷量が大幅増加

| 戦略 | 取引券レート | 電力収支 |
|---|--:|--:|
| A: HC単独 | 2,170 | +1 |
| B: SC依存 | 成立せず | - |
| C: LC単独 | 2,306 | -7 |
| **D: HC+LC混合** | **2,443** | **+143** |

### 最適化のポイント

1. **鉄製部品を廃止**: 1.00券/青鉄は非効率（カプセル/缶詰は1.75券/青鉄）
2. **全鉱石100%消費**: 余剰ゼロで最大効率
3. **電力バッテリーを最小化**: 必要な16/minのみ生産、残り32/minを出荷
4. **高効率製品に集中**: カプセルⅢ/缶詰Ⅲで青鉄鉱全量を消費

## 備考

### 四号谷地

- **全鉱石100%消費**（最適化により余剰ゼロを達成）
- **HC+LC混合戦略が最適** - HCは電力効率が高いため電力専用、LCは全量出荷
- **0.25刻み最適化** - 発電機の0.25個刻み消費を活用し、HC1.5/minを追加出荷
- 鉄製部品は非効率（1.00券/青鉄）のため採用せず
- カプセルⅢ/缶詰Ⅲは高効率（1.75券/青鉄）で青鉄鉱の大半を消費
- 電力バッファ2,000 unit/sec確保済み（+143の安全余裕）

### 武陵

- 息壌は全量をバッテリー原料に（売却より5倍効率的）
- 取引券蓄積レートが律速のため、生産能力の約78%しか実効化できない
- 防衛任務・オペレータ派遣の最適化で実効レート向上の余地あり
