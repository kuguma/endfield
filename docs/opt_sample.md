## 最適解との比較

| 項目 | 現行 | 最適解 | 差分 |
|---|--:|--:|--:|
| 取引券レート | 2,175/min | 2,443/min | **+268 (+12.3%)** |
| 源石鉱利用率 | 96.4% | 100% | +3.6% |
| 紫晶鉱利用率 | 100% | 100% | - |
| 青鉄鉱利用率 | 102.8% | 100% | -2.8% |
| 青鉄鉱過不足 | -30/min | 0 | **要修正** |
| 電力消費 | 3,235 unit/sec | 1,157 unit/sec | **-2,078 (64%削減)** |
| 電力バッファ | 不明 | 2,000 unit/sec | +143余裕 |

### 主な非効率要因

1. **青鉄鉱が30/min不足** - 現行構成は持続不可能
2. **低効率製品の生産**:
   - 鉄製部品: 1.00券/青鉄（最適解では廃止）
   - 鋼製部品: 1.50券/青鉄（低効率）
   - 缶詰Ⅱ/カプセルⅡ: 1.35券/青鉄（中程度）
3. **高効率製品の過少生産**:
   - 缶詰Ⅲ/カプセルⅢ: 1.75券/青鉄（最高効率）を各6/minしか生産していない
4. **源石鉱20/min余剰** - 結晶外殻増産で活用可能

### 電力消費差異の分析

現行ポートフォリオは最適解より2,078 unit/sec多く電力を消費している。

| 要因 | 現行 | 最適解 | 差分 |
|---|--:|--:|--:|
| 大容量バッテリー（2台 vs 1台） | 1,030 | 398 | +632 |
| 中容量バッテリー（1台 vs 0台） | 195 | 0 | +195 |
| カプセル/缶詰Ⅰ・Ⅱ（4製品 vs 0製品） | 930 | 0 | +930 |
| 低効率製品（鉄製部品・鋼製部品等） | 158 | 42 | +116 |
| 小容量バッテリー（1台 vs 6.33台） | 78 | 349 | -271 |
| カプセルⅢ・缶詰Ⅲ（2台 vs 4台） | 824 | 368 | +456 |
| **合計** | **3,235** | **1,157** | **+2,078** |

**主要因**:
1. **カプセル/缶詰Ⅰ・Ⅱの生産**: +930 unit/sec（最大要因）
   - 中間グレード製品は電力効率が悪く、最適解では廃止
2. **大容量バッテリーの過剰生産**: +632 unit/sec
   - 2台運用だが電力用に必要なのは実質1台未満
3. **カプセルⅢ・缶詰Ⅲの過少生産**: +456 unit/sec
   - 同じ生産量でも製造チェーンの効率差がある

### 改善案

青鉄鉱の制約内で最大化するには:
- 鉄製部品・鋼製部品・結晶外殻を廃止
- 缶詰Ⅰ/Ⅱ・カプセルⅠ/Ⅱを全廃止（電力効率が悪い）
- 缶詰Ⅲ/カプセルⅢを増産（各12/min）
- **HC+LC混合戦略を採用**:
  - 大容量バッテリー6/min生産、うち4.5/minを電力用、1.5/minを出荷
  - 小容量バッテリー38/minを全量出荷
  - 残り紫晶鉱50/minは紫晶部品で消費
- **期待効果**: 2,175 → 2,443 券/min（+12.3%）、電力消費64%削減
