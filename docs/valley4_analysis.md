# 四号谷地 出荷製品の効率分析

四号谷地（Valley IV）で拠点取引可能な全14製品について、原材料コストと電力コストを考慮した実効利益率を分析する。

## 結論

| 目的 | 推奨製品 | 理由 |
|---|---|---|
| 利益率重視 | **中容量バッテリー** | 実効利益率 +13.8% で最高 |
| 利益総額重視 | **大容量バッテリー** | 実効利益 +32.0/min で最高 |
| 避けるべき | カプセルⅠ / 缶詰Ⅰ | 原価割れ −24.1% |

## 分析の前提

### 生産速度

| サイクル | マシン例 | 生産速度 |
|---|---|---|
| 2秒 | Refining / Moulding / Fitting | 30個/min |
| 10秒 | Filling / Packaging | 6個/min |

### マシン電力

| マシン | 消費電力 |
|---|--:|
| Shredding Unit | 5 |
| Refining Unit | 5 |
| Moulding Unit | 10 |
| Fitting Unit | 20 |
| Grinding Unit | 50 |
| Filling Unit | 20 |
| Packaging Unit | 20 |

### 電力コストの算出方法

電力の機会コストを「HC Valley Battery を売った場合の逸失利益」として計算する。

- **HC Valley Battery**: 1,100 power × 40 sec = 44,000 unit·sec
- **HC Battery 取引券価値**: 70
- **電力単価**: 70 ÷ 44,000 ≈ **0.00159 券/unit·sec**

| 項目 | 計算式 |
|---|---|
| チェーン電力 | フルレート生産に必要な全マシンの消費電力合計 |
| エネルギー/個 | チェーン電力 × 60 ÷ 生産速度 |
| 電力コスト | エネルギー/個 × 70 ÷ 44,000 |
| 総コスト | 原価 + 電力コスト |
| 実効利益 | 単価 − 総コスト |
| 実効利益率 | 実効利益 ÷ 総コスト |

## 考察

### 電力コストの影響

| 製品 | 電力なし利益率 | 実効利益率 | 低下幅 |
|---|--:|--:|--:|
| 蕎花カプセルⅢ | +17% | +4.0% | −13pt |
| 大容量バッテリー | +20% | +8.2% | −12pt |
| 中容量バッテリー | +20% | +13.8% | −6pt |

Tier III 製品は Grinding Unit（50 power）を多用するため、電力コストが利益を大きく侵食する。

### なぜ中容量バッテリーが効率的か

- **Grinding Unit を使わない**: Ferrium → Ferrium Part の直接変換
- **マシン台数が少ない**: チェーン電力 85（大容量の 21%）
- **原材料効率も良い**: 源石鉱 + 青鉄鉱の2種のみ

### 避けるべき製品

- **カプセルⅠ / 缶詰Ⅰ**: 原価 12.5 に対し単価 10。Amethyst Bottle（紫晶鉱 2個分）を 5個消費するコストが重い
- **基礎素材（結晶外殻/紫晶ボトル/部品）**: 電力分だけ赤字。ただし他製品の中間材料として必要
